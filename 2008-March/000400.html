On Friday 28 March 2008 23:57:55 Alexandru Tudose wrote:
> Hello,
>
>  Is there a way to make Download To ---> Browse menu to bypass default
> download directory. It's always a pain to go in Preferences dialog and
> change default directory, when I want to download a file to a different
> partion.
>
> Thanks.

If you mean unfinished dir then no. Files are first downloaded to the 
unfinished folder and after that moved to the destination dir if it has free 
space. ^w^w^w... Actually, I'm not sure how it behaves if you leave the 
unfinished dir empty. If I remember correctly older DC++ versions downloaded 
directly to the destination in case of empty unfinished  dir. Current 
versions might just default to the home dir/.dc++ if empty. So you could try 
that and see what happens. 

If it's not the unfinished dir what you're after then I'm not sure if I 
understand the question. Because you don't have to change the download dir to 
dl to a different partition. You can browse for the destination or set-up 
predefined alternative locations for download.
