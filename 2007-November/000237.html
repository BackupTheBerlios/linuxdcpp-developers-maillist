Hi,

this patch (attached) makes it possible to open finished files through 
the default application specified in your favourite desktop environment. 
(In Windows this would be called open by association or something like 
that.)

I'm not satisfied with it yet, but it works. Right now the catch-all 
script 'xdg-open' is hard coded into Wulfor::openURI. I'm open for 
suggestions.

As for error handling, I'm going with a dialog on error. This seems to 
be the first instance of showing a error message dialog in the whole 
program? I'm having trouble accessing the mainwindow pointer, which is 
proper to set when creating modal dialogs. I guess I'm open to 
suggestions on this one too :)

The patch is work-in-progress.
