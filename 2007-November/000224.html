Details: From Changelog:

Implementation of this feature actually breaks stuff: LinuxDC++ no longer obeys the same setting I've set in Gnome. This sucks, because I certainly don't want to configure toolbar style for every application I use, and I want at least GTK-based applications to follow the setting I set in a GTK-based environment.

I don't mind LDC++ having this feature, but in such case a "default" setting should be available and defaulted to.
